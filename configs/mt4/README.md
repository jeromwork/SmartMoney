# MT4 Config Templates

- `backtest.ini.template` используется как база для CLI-бэктестов
- `demo.ini.template` используется как база для запуска demo-инстанса
- `demo.tpl` не создается автоматически, потому что корректный шаблон удобнее сохранить из уже настроенного терминала

После первого ручного запуска `mt4-demo` сохраните нужный шаблон графика в каталоге:

`terminals/mt4-demo/templates/demo.tpl`
