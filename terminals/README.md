# Terminal Layout

- `mt5-dev` — ручная отладка, загрузка истории, проверка графиков
- `mt5-test` — CLI-бэктесты
- `mt5-demo` — demo-forward test

Эти каталоги намеренно разделены, чтобы Codex мог безопасно запускать тесты и demo без смешивания состояния.

Примечание: все каталоги терминалов и pipeline используют только `MT5/MQL5`.
