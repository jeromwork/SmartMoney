# SmartMoney MQL4 Workspace

Изолированная среда для разработки `MQL4`-советников и индикаторов в `VSCode + Codex` с отдельными инстансами `MT4` под `dev`, `test`, `demo`.

## Что уже подготовлено

- Структура проекта для `Experts`, `Indicators`, `Include`
- Черновик стратегии в `docs/strategy`
- PowerShell-скрипты для `build`, `backtest`, `demo-launch`, `collect-logs`, `parse-report`, `sync-data`
- Каркасы `EA`, индикатора и общих утилит
- Конфигурация `VSCode tasks`
- Локальный Git-репозиторий

## Что осталось, чтобы сборка и тесты стали рабочими

1. Положить брокерский инсталлятор в `downloads/finam5setup.exe`
2. Запустить `scripts/install-finam-terminal.ps1`
3. Убедиться, что установлен именно `MT4` с каталогом `MQL4`, а не `MT5/MQL5`
4. При необходимости открыть `mt4-dev` и подключить demo/real сервер брокера для подкачки истории
5. После этого использовать VSCode tasks или PowerShell-скрипты напрямую

## Рабочий цикл

1. Формализовать стратегию в `docs/strategy/current-strategy.md`
2. Дорабатывать `src/MQL4/...`
3. Запускать `build`
4. Запускать `backtest`
5. Анализировать `logs/` и `reports/`
6. После стабилизации запускать `mt4-demo`

## Важное ограничение

Сейчас в проекте нет установленного терминала и нет инсталлятора внутри `downloads/`. Скрипты готовы, но без этого нельзя выполнить реальную компиляцию и бэктест.
