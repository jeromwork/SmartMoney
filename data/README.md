# Data Layout

- `data/broker/` — служебная выгрузка, если понадобится сохранить данные из подключенного терминала отдельно
- `data/import/<SYMBOL>/` — импортируемая история по символам
- `data/ticks/<SYMBOL>/` — тиковые файлы для более точного тестирования

Ожидаемый процесс:

1. Подключить `mt4-dev` к брокеру
2. Дождаться подкачки истории
3. При необходимости сохранить повторяемый набор данных в `data/import` или `data/ticks`
4. Запустить `scripts/sync-data.ps1`

Примечание: каталог слота называется `mt4-dev` исторически, но используется с `MT5`.
